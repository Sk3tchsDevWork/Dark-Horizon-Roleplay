Installation:
1) Insert folder "bzzz_recycler" to resources folder
2) Add to server.cfg
3) Restart server



If you stream props in another resource, you must edit the fxmanifest.
Add this line:
data_file 'DLC_ITYP_REQUEST' 'stream/bzzz_prop_recycler_a.ytyp'

Then the server must be restarted. 
YTYP loads properties of props.
